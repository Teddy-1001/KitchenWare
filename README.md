# KitchenWare

KitchenWare is a modern e-commerce platform designed for selling kitchen and home essentials online. The application provides a seamless shopping experience with product browsing, cart management, user authentication, and order processing.

## Features

* User registration and authentication
* Secure password hashing with bcrypt
* Product catalog and product management
* Shopping cart with quantity controls
* Session-based cart persistence
* Customer order management
* Admin account support
* Responsive user interface built with Tailwind CSS
* PostgreSQL database integration
* Server-side rendering with EJS templates

## Tech Stack

### Backend

* Node.js
* Express.js
* PostgreSQL
* Express Session
* bcrypt

### Frontend

* EJS
* Tailwind CSS
* HTML5
* JavaScript

## Project Goals

KitchenWare aims to provide a scalable foundation for an online kitchenware store while demonstrating full-stack web development concepts including authentication, database management, session handling, and e-commerce workflows.
